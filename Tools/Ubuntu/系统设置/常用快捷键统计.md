# 1. Ubuntu 系统本身快捷键
* alt+table 切换应用
* ctrl + r +  上下键 且含workspace（可以快速回到桌面）
* windows + 上下左右调节当前软件大小和位置

# 2. teminator
*   alt + 上下左右            切换选中shell 框
*   ctrl+shift+上下左右     调节shell框大小

# 3. vscode
*  ctrl+ 加减        调节字体大小

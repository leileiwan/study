# 2. 声明
## 2.1 声明的通用形式
```
var name type = value
```
* 类型和变量值可以省略一个
* 包级别初始化在main开始之前进行，局部变量的声明和初始化在函数执行期间执行
* var 声明的使用场景
    * 声明类型和value类型不一致情况
        ```
        var myValue float64 = 100
        ```
    * 赋值的value对声明的变量不重要情况
        ```
        var myValue float64
        ```

## 2.2 短变量声明
```
name:=value
```
* 短声明变量不需要声明左边所有的变量，但是至少要声明一个变量，其余变量作为赋值使用

# 2. group
* 创建group
    * groupadd  test
* 修改group
    * groupmod -n test2  test（将test组的名子改成test2）
* 删除组
    * groupdel test2
* 查看组
    * groups （查看当前用户所在的组）
    * groups apacheuser （查看指定用户）
    * cat /etc/group （查看所有组）
